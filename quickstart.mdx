Sign up for Komodo AI, connect your cloud account, and start running your workloads in minutes.

### 1. Create an account and connect your cloud

Before you get started, make sure to create an account and connect your cloud account(s).

1. [Sign up](https://app.komodoai.dev) for a free account at [app.komodoai.dev](https://app.komodoai.dev) and log in.
2. Connect a cloud provider on the [Settings](https://app.komodoai.dev/settings) page via the provider-specific workflow.

<img src="images/settings-cloud-connect-beacon.png" />

### 2. Install Komodo CLI and authenticate

Install the CLI for interacting with the Komodo API:

```bash
pip install komo
```

Head to the [Settings](https://app.komodoai.dev/settings) page and copy your Komodo API key.

<img height="400" src="images/settings-api-key.png" />
Next, log into Komodo AI

```bash
komo login
```

Provide your API key when prompted.
