title: register
---

Create a new Komodo account.

You will be prompted to enter your email and choose a password.
After creating an account, you'll want to then [create an API key](./api-key/create).

### Usage
`komo register [OPTIONS]`

### Options

| Option                              	| Description                                                                                    | Type |
|-------------------------------------	|----------------------------------------------------------------------------------------------- |----- |


### Examples

Create a new account
```bash
komo register
```
