Initialize a new Komodo project in the current directory.

### Usage

`komo init project [OPTIONS]`

### Options

| Option                              	| Description                                                                                    | Type |
|-------------------------------------	|----------------------------------------------------------------------------------------------- |----- |

### Examples

Initialize a new project in the current directory
```bash
komo init project
```

This will create a template project configuration in  `.komo/project.yaml`.
In this configuration file, there will be different configurations for the different backends you created earlier.
Edit this file to fit your needs (required and recommended fields to edit are marked with `TODO`).
