---
title: job
---
Initialize a new Komodo job file for the current project.

### Usage

`komo init job [OPTIONS]`

### Options

| Option                              	| Description                                                                                    | Type |
|-------------------------------------	|----------------------------------------------------------------------------------------------- |----- |
| &#x2011;n, &#x2011;&#x2011;name  	| The name of job task to create. Required.   | Text 	|


### Examples

<Note>This will create a template job configuration in  `my-job.yaml`. You can then edit this file according to your needs.</Note>
```bash
komo init job -n my-job
```
