---
title: task
---
Initialize a new Komodo task file for the current project.

### Usage

`komo init task [OPTIONS]`

### Options

| Option                              	| Description                                                                                    | Type |
|-------------------------------------	|----------------------------------------------------------------------------------------------- |----- |
| &#x2011;n, &#x2011;&#x2011;name  	| The name of the task to create. Required.   | Text 	|


### Examples

Create a task named `my-task`
<Note>This will create a template task configuration in  `.komo/my-task.yaml`. You can then edit this file according to your needs.</Note>
```bash
komo init task -n my-task
```
