---
title: shell
---
Open a shell into a running job.

### Usage

`komo shell [OPTIONS] JOB_ID`

### Options

| Option                              	| Description                                                                                                                                                                                             | Type |
|-------------------------------------	|---------------------------------------------------------|----- |
| &#x2011;i, &#x2011;&#x2011;node-index | The index of the node to get logs from. Defaults to 0.  | Int  |

### Examples

Open a shell into job `abcdef123456`
```bash
komo shell abcdef123456
```

Open a shell into the second node of job `abcdef123456`
```bash
komo shell -i 1 abcdef123456
```
