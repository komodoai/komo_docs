## api-key create

The `api-key create` command is used to log into Komodo and retrieve your API key. This key is required to use Komodo's services. To retrieve your API key, simply run the following command:

```
komo api-key create
```

You will be prompted to enter your Komodo username (email) and password. Once authenticated, your API key and the instructions on where to place it will be displayed in the output.
