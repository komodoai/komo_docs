Create an API key.

### Usage

`komo api-key create [OPTIONS]`

### Options

| Option                              	| Description                                                                                    | Type |
|-------------------------------------	|----------------------------------------------------------------------------------------------- |----- |

### Examples

Create an API key
<Note>You will be prompted to enter your Komodo username (email) and password. The newly created API key will be stored in `~/.komo/api-key`</Note>
```bash
komo api-key create
```
