title: create

<Info>Currently, only AWS EKS clusters are supported with this command</Info>

Create the compute infrastructure needed to run Komodo jobs using Terraform.
This is provided as a convenient alternative to setting up your infrastructure manually.

Once you do this, you can add [node groups](./add-node-group) and [storage](./add-storage).


### Usage

`komo compute create [OPTIONS]`

### Options

| Option                              	    | Description                                                     	| Type 	|
|-----------------------------------------  |-----------------------------------------------------------------	|------	|
| &#x2011;n, &#x2011;&#x2011;name 	        | The name for the newly created compute infrastructure. Required.  | Text 	|
| &#x2011;t, &#x2011;&#x2011;compute-type  	| The type of compute infrastructure to create. Defaults to eks.    | [eks] |
| &#x2011;m, &#x2011;&#x2011;manual-confirm | Manually confirm terraform plan.                                  |      	|


### Examples

Create an AWS EKS cluster named `my-cluster`
```bash
komo compute create -t eks -n my-cluster
```
