title: create

<Info>Currently, only AWS compute types are supported with this command</Info>

Create the compute infrastructure needed to run Komodo jobs using Terraform.

### Usage

`komo compute create [OPTIONS]`

### Options

| Option                              	    | Description                                                     	                     | Type 	|
|-----------------------------------------  |-----------------------------------------------------------------	                     |------	|
| &#x2011;n, &#x2011;&#x2011;name 	        | The name for the newly created compute infrastructure. Required.                       | Text 	|
| &#x2011;&#x2011;aws-region 	| The AWS region to use. Required when using `aws` for `compute-type`.                   | Text 	|
|  &#x2011;&#x2011;aws-profile 	| The AWS profile to use (eg. `default`). Required when using `aws` for `compute-type`.  | Text 	|
| &#x2011;t, &#x2011;&#x2011;compute-type  	| The type of compute infrastructure to create. Defaults to aws.                         | [aws] |
| &#x2011;m, &#x2011;&#x2011;manual-confirm | Manually confirm terraform plan.                                                       |      	|


### Examples

Setup an AWS environment named `my-cluster` in `us-east-1`
```bash
komo compute create -t aws -n my-cluster --aws-region us-east-1 --aws-profile default
```
