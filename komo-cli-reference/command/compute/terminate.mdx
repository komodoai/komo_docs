---
title: terminate
---

Terminate infrastructure created with `komo compute create`.

### Usage

`komo compute terminate [OPTIONS]`

### Options

| Option                              	    | Description                                                     	| Type 	|
|-----------------------------------------  |-----------------------------------------------------------------	|------	|
| &#x2011;n, &#x2011;&#x2011;name 	        | The name of compute infrastructure to destroy. Required.  | Text 	|
| &#x2011;t, &#x2011;&#x2011;compute-type  	| The type of compute infrastructure to destory. Defaults to eks.   | [eks] |
| &#x2011;m, &#x2011;&#x2011;manual-confirm | Manually confirm terraform plan.                                  |      	|

### Examples

Destroy the EKS cluster named `my-cluster`
```bash
komo compute terminate -n my-cluster -t eks
```
