## compute add-storage

The `compute add-storage` command is used to add a storage provider to the Komodo compute cluster.

### Usage

```
komo compute add-storage [OPTIONS]
```

### Options

```
-s, --storage-type: Storage type to add.
```

```
--s3-bucket: (required) Name of the S3 bucket backing the storage.
```

Supported storage types: `fsx`
