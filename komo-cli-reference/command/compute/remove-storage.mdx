### compute remove-storage

The `compute remove-storage` command is used to remove a storage provider from the Komodo compute cluster.

```
komo compute remove-storage [OPTIONS]
```

```
-s, --storage-type: Storage type to remove.
```

```
--s3-bucket: (required) Name of the S3 bucket backing the storage.
```

Supported storage types: `fsx`
