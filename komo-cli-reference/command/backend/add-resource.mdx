title: add-resource
---

Add a resource to an existing backend.<br/><br/>
You will be prompted to enter information about this resource (eg. instance type).
The information you'll be prompted to enter depend on the type of backend that you're adding the resource to.

### Usage

`komo backend add-resource [OPTIONS]`

### Options

| Option                              	| Description                                                                                                                                                       	| Type 	|
|-------------------------------------	|------------------------------------------------------ |------	|
| &#x2011;b, &#x2011;&#x2011;backend  	| Name of the backend to add the resource to. Required. | Text 	|
| &#x2011;n, &#x2011;&#x2011;name 	    | Name of the resource to create. Required.             | Text 	|

### Examples

Create a resource named `g4` to the backend named `aws`

<Note>You will be prompted to enter details about this resource.</Note>
```bash
komo backend add-resource -b aws -n g4
```
