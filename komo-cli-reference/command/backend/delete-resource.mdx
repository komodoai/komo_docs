---
title: delete-resource
---

Delete a resource for an existing backend.

### Usage

`komo backend delete-resource [OPTIONS]`

### Options

| Option                              	| Description                                                                                                                                                       	| Type 	|
|-------------------------------------	|----------------------------------------------------------- |------	|
| &#x2011;b, &#x2011;&#x2011;backend  	| Name of the backend to delete the resource from. Required. | Text 	|
| &#x2011;r, &#x2011;&#x2011;resource 	| Name of the resource to delete. Required.                  | Text 	|

### Examples

Delete the resource named `g4` from the backend named `aws`
```bash
komo backend delete-resource -b aws -r g4
```
