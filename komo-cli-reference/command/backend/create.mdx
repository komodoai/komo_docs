Create a Komodo backend for your existing compute infrastructure.<br/><br/>
You will be prompted to enter the information needed to connect to your infrastructure.
The exact information you'll be prompted to enter depends on the type of backend you're creating.

### Usage

`komo backend create [OPTIONS]`

### Options

| Option                              	    | Description                                        | Type 	|
|-----------------------------------------  |--------------------------------------------------- |------	|
| &#x2011;n, &#x2011;&#x2011;name 	        | The name for the newly created backend. Required.  | Text 	|
| &#x2011;t, &#x2011;&#x2011;backend-type  	| The type of backend to create. Required.           | [local \| kubernetes \| aws-batch] |

### Examples

Create an AWS Batch backend named `aws`

<Note>You will be prompted to enter information specific to this type of backend</Note>
```bash
komo backend create -t aws-batch -n aws
```
