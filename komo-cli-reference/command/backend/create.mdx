### backend create

The `backend create` command is used to create a backend entity on the API server. A backend entity can then be used to run your jobs.

### Usage

```
komo backend create [OPTIONS] NAME
```

### Options

```
Options:
  -t, --type [local|kubernetes]  Type of backend to create.  [required]
```

You will be prompted for backend details.
