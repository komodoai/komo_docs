List all of your backends.

### Usage

`komo backend list [OPTIONS]`

### Options

| Option                              	    | Description                                        | Type 	|
|-----------------------------------------  |--------------------------------------------------- |------	|
| &#x2011;c, &#x2011;&#x2011;configs 	    | Display the config for each backend.               |       	|


### Examples

List all of your backends
```bash
komo backend list
```
