With the Komodo CLI, you can interact with Komodo's platform to easily manage your machines, jobs, and services.

## Usage

`komo [COMMAND] [OPTIONS] [ARGS]...`

## Commands

| Command | Description            |
| ------- | ---------------------- |
| job     | Manage Komodo jobs     |
| machine | Manage Komodo machines |
| service | Manage Komodo services |
