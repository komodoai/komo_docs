title: ssh
---

SSH into a running job.

### Usage

`komo job ssh [OPTIONS] JOB_ID`

### Options

| Option                                | Description                                            | Type |
| ------------------------------------- | ------------------------------------------------------ | ---- |
| &#x2011;i, &#x2011;&#x2011;node-index | The index of the node to get logs from. Defaults to 0. | Int  |

### Examples

SSH into job `abcdef123456`

```bash
komo job ssh abcdef123456
```

SSH into the second node of job `abcdef123456`

```bash
komo job ssh -i 1 abcdef123456
```
