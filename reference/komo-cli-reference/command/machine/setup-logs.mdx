---
title: setup-logs
---

Get the setup logs for a given machine.

## Usage

`komo machine setup-logs [OPTIONS] MACHINE_NAME`

## Options

| Option                            | Description                                                                                                                                                                    | Type |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---- |
| &#x2011;w, &#x2011;&#x2011;follow | If this option is specified, this command will continuously output new log entries until the job finishes. Otherwise, this command will output the current logs and then exit. | flag |

## Examples

Get the logs for machine `dev`

```bash
komo machine setup-logs -f dev
```
