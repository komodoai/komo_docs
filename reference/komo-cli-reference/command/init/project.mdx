Initialize a new Komodo project in the current directory.

### Usage

`komo init project [OPTIONS]`

### Options

| Option                              	| Description                                                                                    | Type |
|-------------------------------------	|----------------------------------------------------------------------------------------------- |----- |

### Examples

Initialize a new project in the current directory
```bash
komo init project
```

This will create a template project configuration in  `komoproj.yaml`.
In this configuration file, you can set different options that will apply to all your jobs/machines/served models for this project.
